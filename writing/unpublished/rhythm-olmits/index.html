Rhythm OLMITS

Generated as part of my computer science Masters coursework was an intelligent tutoring system teaching playing rhythm from music notation with an open learner model supporting learner metacognition. It was implemented as an Electron app with a GUI developed using React and a libpd audio engine. The codebase for the ITS can be found here on Github and a link to the paper can be found on Georgia Tech's SMARTech page.
